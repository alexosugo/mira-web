# Mira Web

Marketing landing page for Mira: answers a shop's Instagram DMs and sells in them (Kenyan Instagram sellers, waitlist conversion). Vite + React 18 + TypeScript + Tailwind, deployed on Netlify.

## Design Context

Read these before any UI, copy, or design work:

- **`PRODUCT.md`**: register (brand), users, positioning (Instagram-DM commerce, not multi-channel support), anti-references, strategic design principles.
- **`DESIGN.md`**: visual system spec (warm editorial calm: paper/ink/clay palette, Fraunces / Instrument Sans / IBM Plex Mono, named rules like The Clay Rule).
- **`docs/voice-and-tone.md`**: voice rules. No jargon, no "AI-powered", no exclamation points; say what Mira does.

Note: the README's multi-channel framing (WhatsApp/Facebook) is legacy; PRODUCT.md's Instagram-DM commerce positioning wins.
